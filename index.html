String Methods

Methods are functions associated with an object or class in Python Programming. They are used to perform specific tasks or operations on the data stored within objects or classes.

In Python, all variables are objects. We will learn about the class in upcoming classes.

In simple words, methods are actions that an object can perform. You use a method by attaching it to an object with a dot (.).

Different objects have different methods that can be performed on them.
